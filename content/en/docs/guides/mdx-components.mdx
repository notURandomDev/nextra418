# MDX Components

Nextra supports using React components directly in Markdown. Nextra418 includes several common built-in components.

## Common Component Examples

<Tabs items={["Callout", "Tabs", "Cards"]}>
  <Tabs.Tab>
    ```mdx
    import { Callout } from "nextra/components";

    <Callout type="info">This is an info message.</Callout>
    <Callout type="warning">This is a warning message.</Callout>
    <Callout type="error">This is an error message.</Callout>
    ```
    <Callout type="info">This is an info message.</Callout>
    <Callout type="warning">This is a warning message.</Callout>
    <Callout type="error">This is an error message.</Callout>

  </Tabs.Tab>
  <Tabs.Tab>
    ```mdx
    import { Tabs } from "nextra/components";

    <Tabs items={["npm", "pnpm"]}>
      <Tabs.Tab>npm install</Tabs.Tab>
      <Tabs.Tab>pnpm install</Tabs.Tab>
    </Tabs>
    ```

    <Tabs items={["npm", "pnpm"]}>
      <Tabs.Tab>npm install</Tabs.Tab>
      <Tabs.Tab>pnpm install</Tabs.Tab>
    </Tabs>

  </Tabs.Tab>
  <Tabs.Tab>
    ```mdx
    import { Cards } from "nextra/components";

    <Cards>
      <Cards.Card title="Introduction" href="/docs/introduction" />
      <Cards.Card title="Visit Official Site" href="https://nextra.site" arrow />
    </Cards>
    ```
    <Cards>
      <Cards.Card title="Introduction" href="/docs/introduction" />
      <Cards.Card title="Visit Official Site" href="https://nextra.site" arrow />
    </Cards>

  </Tabs.Tab>
</Tabs>

Refer to the [Nextra official documentation](https://nextra.site/docs/built-ins) for more component usage.
