# Introduction

**Nextra418** is shorthand for Nextra 4 + i18n—a minimal documentation template built on [Nextra 4](https://nextra.site/) with out-of-the-box i18n configuration.

Its design philosophy is **Lean and Production-Ready**. It strips away complex, redundant code and keeps only the core documentation features, so you can focus on content creation.

## Why Choose Nextra418?

### 🚀 Latest Tech Stack

Built on **Next.js 16 (App Router)** and **Nextra 4**, giving you immediate access to the powerful performance of React Server Components (RSC).

### 🌍 Out-of-the-Box i18n

Nextra418 follows the **Dictionary** approach recommended by Nextra 4 and streamlines it to the core configuration. This makes multilingual UI copy management more standardized while keeping the template lightweight.

### 🔍 Lightning-Fast Search

Integrated **Pagefind**—a Rust-based static search engine that delivers millisecond-level full-text search on the client side, no backend required.

### 🎨 Flexible Styling

Maintains an **unopinionated** design philosophy—no forced CSS framework bundling. Integrate your preferred styling solution based on project needs.

## Difference from Official Template

The official [nextra-docs-template](https://github.com/shuding/nextra-docs-template) remains stuck in the Next.js 13 + Nextra 2 era and has seen little maintenance.

Nextra418 fills this gap, providing you with a modern, production-ready starting point.
