# 核心配置

Nextra418 将核心配置集中在 `app/_config` 和 `app/_dictionaries` 目录下，让你能轻松定制站点。

## 布局配置

`app/_config/layout.config.tsx` 文件控制着站点的整体布局，包括导航栏、侧边栏、页脚等。

### 核心选项

- **docsRepositoryBase**: 文档仓库的基础 URL，用于生成“编辑此页”链接。
- **navbar**: 顶部导航栏配置，包括 Logo 和项目链接。
- **footer**: 页脚内容配置。
- **toc**: 右侧目录（Table of Contents）配置。
- **sidebar**: 左侧侧边栏配置。

```tsx filename="app/_config/layout.config.tsx"
export const getLayoutConfig = (dictionary: Dictionary) => ({
  // ...
  docsRepositoryBase: "https://github.com/your-name/your-repo/blob/main",
  navbar: (
    <Navbar
      logo={<span>My Docs</span>}
      projectLink="https://github.com/your-name/your-repo"
    />
  ),
  // ...
});
```

## SEO 配置

`app/_config/meta.config.tsx` 文件用于管理站点的元数据（Metadata），这对 SEO 至关重要。

你可以在这里修改站点的标题、描述、关键词以及 Open Graph 图片等信息。

```tsx filename="app/_config/meta.config.tsx"
export const getMetadata = (): Metadata => ({
  title: {
    default: "My Docs",
    template: "%s | My Docs",
  },
  description: "A minimal documentation site.",
  // ...
});
```

## 多语言配置 (i18n)

Nextra418 内置了基于"字典"的多语言支持方案，解决了 Nextra 原生 i18n 无法覆盖 UI 组件文案的问题。

以添加**西班牙语（Español）**为例，配置步骤如下：

<Steps>
### 定义语言

在 `app/_dictionaries/i18n-config.ts` 中添加新语言：

```ts {3} filename="app/_dictionaries/i18n-config.ts"
const i18n = {
  defaultLocale: "en",
  locales: ["en", "zh", "es"], // 添加 "es" 代表西班牙语
} as const;
```

### 添加字典文件

在 `app/_dictionaries` 目录下创建 `es.ts`，参照 `en.ts` 的结构填入翻译内容：

```ts filename="app/_dictionaries/es.ts"
export default {
  backToTop: "Volver arriba",
  editPage: "Editar esta página en GitHub",
  search: {
    placeholder: "Buscar contenido...",
  },
  feedback: {
    content: "¿Tienes preguntas? Danos tu opinión",
  },
  toc: {
    title: "En esta página",
  },
  release: {
    title: "Lanzado",
    notes: "Ver las notas de lanzamiento",
  },
  lastUpdated: "Última actualización el",
};
```

### 更新布局配置

在 `app/_config/layout.config.tsx` 的 `i18n` 选项中添加西班牙语，这样语言切换下拉框中就会多出一个"Español"选项：

```tsx {4} filename="app/_config/layout.config.tsx"
i18n: [
  { locale: "zh", name: "简体中文" },
  { locale: "en", name: "English" },
  { locale: "es", name: "Español" }, // 添加西班牙语
],
```

### 创建内容目录

在 `content` 目录下创建对应的语言目录结构，并创建文档内容：

```text {6-8}
content/
├── zh/
│   └── docs/
├── en/
│   └── docs/
└── es/              # 新建西班牙语目录
    └── docs/
        └── introduction.mdx
```

</Steps>
