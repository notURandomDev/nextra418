# 布局配置

`app/_config/layout.config.tsx` 文件控制着站点的整体布局，包括导航栏、侧边栏、页脚等。

### 核心选项

- **docsRepositoryBase**: 文档仓库的基础 URL，用于生成"编辑此页"链接。
- **navbar**: 顶部导航栏配置，包括 Logo 和项目链接。
- **footer**: 页脚内容配置。
- **toc**: 右侧目录（Table of Contents）配置。
- **sidebar**: 左侧侧边栏配置。

```tsx filename="app/_config/layout.config.tsx"
export const getLayoutConfig = (dictionary: Dictionary) => ({
  // ...
  docsRepositoryBase: "https://github.com/your-name/your-repo/blob/main",
  navbar: (
    <Navbar
      logo={<span>My Docs</span>}
      projectLink="https://github.com/your-name/your-repo"
    />
  ),
  // ...
});
```
